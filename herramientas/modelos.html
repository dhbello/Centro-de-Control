Modelo GEOCOL 2004

Este modelo fue desarrollado con el fin de representar geométricamente una aproximación fisicomatemática del campo de gravedad y su potencial gravitatorio en Colombia.

Detalle
  • Permite calcular las ondulaciones geoidales para Colombia.
  • Desarrollado mediante la combinación de información gravimétrica a diferentes resoluciones.
  • Generado con datos gravimétricos de diferentes contribuidores nacionales e internacionales.

Declinación Magnética.

Este modelo fue adaptado para calcular la declinación magnética de una coordenada geográfica ubicada en el territorio colombiano.

Detalle
  • Utiliza los parámetros de cálculo del Modelo Magnético Mundial (WMM)
  • La calculadora proporciona datos de manera rápida acorde a la coordenada ingresada.
  • No es necesario registrarse para obtener el resultado.
